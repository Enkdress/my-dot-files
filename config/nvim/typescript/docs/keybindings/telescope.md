# Telescope Keybindings

Fuzzy finder for files, text, and more.

| Keybinding | Action | Description |
|------------|--------|-------------|
| `<leader>sf` | Find Files | Search for files in current directory |
| `<leader>sg` | Live Grep | Search for text in all files |
| `<leader>sw` | Search Word | Search for word under cursor |
| `<leader>sh` | Help Tags | Search Neovim help tags |
| `<leader>sk` | Keymaps | Search available keybindings |
| `<leader>sd` | Diagnostics | Search all diagnostics |
| `<leader>sr` | Resume | Resume previous search |
| `<leader>s.` | Recent Files | Search recently opened files |
| `<leader>ss` | Select | Open Telescope picker selector |
| `<leader>/` | Buffer Fuzzy | Fuzzy search in current buffer |
| `<leader>s/` | Open Files | Live grep in open files only |
| `<leader>sn` | Neovim Files | Search Neovim config files |
| `<leader><leader>` | Buffers | Find existing buffers |
