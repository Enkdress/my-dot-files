# LSP Keybindings

Language Server Protocol actions for code navigation and refactoring.

| Keybinding | Action | Description |
|------------|--------|-------------|
| `gd` | Definition | Go to definition of symbol |
| `gD` | Declaration | Go to declaration |
| `gri` | Implementation | Go to implementation |
| `grr` | References | Find all references to symbol |
| `grt` | Type Definition | Go to type definition |
| `gO` | Document Symbols | Fuzzy search symbols in current file |
| `gW` | Workspace Symbols | Fuzzy search symbols in entire project |
| `grn` | Rename | Rename symbol under cursor |
| `<leader>ca` | Code Action | Show available code actions |
| `<leader>th` | Toggle Hints | Toggle inlay hints |
| `<leader>q` | Quickfix | Open diagnostic quickfix list |
| `<leader>e` | Show Diagnostic | Show diagnostic in float |

**Supported Languages:**
- TypeScript / JavaScript
- TSX / React
- JSON
- Astro
- TailwindCSS
- Lua
