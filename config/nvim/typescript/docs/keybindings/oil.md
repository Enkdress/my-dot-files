# Oil Keybindings

File browser that edits directories like buffers.

| Keybinding | Action | Description |
|------------|--------|-------------|
| `<leader>op` | Open Parent | Open parent directory in Oil |
| `-` | Go to parent | Navigate to parent directory (within Oil) |
| `Enter` | Open | Open file/directory |
| `Backspace` | Go to parent | Navigate to parent directory |

**Features:**
- Shows hidden files by default
- Git status indicators
- Integrated with diagnostics and git plugins
- Opens automatically on startup if no files specified
