# General Keybindings

Core Neovim keybindings and editor shortcuts.

## Movement

| Keybinding | Action | Description |
|------------|--------|-------------|
| `<C-h>` | Left Window | Move focus to left window |
| `<C-l>` | Right Window | Move focus to right window |
| `<C-j>` | Lower Window | Move focus to lower window |
| `<C-k>` | Upper Window | Move focus to upper window |

## Basic Editing

| Keybinding | Action | Description |
|------------|--------|-------------|
| `jk` | Escape | Exit insert mode (in insert mode) |
| `<leader>w` | Save | Save current file |
| `<leader>q` | Quit | Close current buffer |
| `<leader>y` | Yank | Yank to system clipboard |
| `<Esc>` | No HL | Clear search highlights |

## Disabled Keys (encourage learning h/j/k/l)

| Keybinding | Action |
|------------|--------|
| `<left>` | Disabled - use h |
| `<right>` | Disabled - use l |
| `<up>` | Disabled - use k |
| `<down>` | Disabled - use j |

## Plugins

| Keybinding | Action | Description |
|------------|--------|-------------|
| `<leader>f` | Format | Format current buffer |
