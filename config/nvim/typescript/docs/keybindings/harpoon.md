# Harpoon Keybindings

Quickly navigate between files in your project.

| Keybinding | Action | Description |
|------------|--------|-------------|
| `<leader>a` | Add file | Add current file to harpoon list |
| `<leader>al` | List files | Open harpoon quick menu |
| `<leader>a1` | Select 1 | Jump to 1st file in list |
| `<leader>a2` | Select 2 | Jump to 2nd file in list |
| `<leader>a3` | Select 3 | Jump to 3rd file in list |
| `<leader>a4` | Select 4 | Jump to 4th file in list |
| `<leader>a5` | Select 5 | Jump to 5th file in list |
| `<leader>ap` | Previous | Jump to previous file in list |
| `<leader>an` | Next | Jump to next file in list |
